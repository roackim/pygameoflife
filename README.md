# pygameoflife
Game of Life implemented in python with the pygame framework

# Dependencies
    python3
    pygame (not pygame-sdl2)

# Run
    python3 main.py

# Controlls
    Enter: pause / resume simulation
    Left  click:  make cell alive
    Right click: make cell dead
    Backspace: make all cells dead
